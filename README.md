# Prework Study Guide Webpage

## Description

This study guide was made for bootcamp students, completing the prework course. It contains notes on HTML, CSS, Git, and JavaScript. While going through this process, I learned mistakes are going to happen and that bug fixing is very important.

## Usage

You can use this prework study guide by reviewing the notes listed in each section. To see what to study first, right-click and open the inspect tab, then click on console and JavaScript can tell you what to check out first.
